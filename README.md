# Go Notes

Goの実装、設計、運用を扱うHugo製の技術ブログです。

## 使い方

```sh
npm install
npm run dev
```

本番ビルドは次のコマンドです。

```sh
npm run build
```

## テーマ

デザインとHugoテンプレートは、MITライセンスの
[wilhelm-murdoch/wilhelm.codes](https://github.com/wilhelm-murdoch/wilhelm.codes)
をスターターとして利用しています。元リポジトリはテーマ単体ではなく完成済みHugoサイトなので、レイアウト・CSS・アセットを取り込み、コンテンツと設定をGoブログ向けに差し替えています。
